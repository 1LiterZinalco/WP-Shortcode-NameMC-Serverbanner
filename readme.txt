== Installation ==

- Sie benötigen eine funktionierende WordPress-Installation.

- Die Shortcode Standard Höhe: 90 Breite: 720

- Shortcode-Server: [mcbanner server="beispiel.de"]
- Shortcode-Höhe/Breite: [mcbanner server="beispiel.de" width="720" height="90"]
